# Implem.Pleasanter
## 概要
軽くて高機能、.NETで動くチケット管理ツール。マネジメントとコミュニケーションに必要となる情報を一元管理することで、情報共有のストレスを軽減し、ビジネスを快適化。商用、非商用を問わず無料で使えるオープンソース・ソフトウェア。  
http://pleasanter.org
